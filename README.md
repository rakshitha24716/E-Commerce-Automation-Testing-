# E-Commerce Automation Testing Suite

![Tests](https://github.com/rakshitha24716/E-Commerce-Automation-Testing-/actions/workflows/test.yml/badge.svg)

Selenium-based automated test suite for e-commerce workflows (login, product search, cart, checkout) built using the Page Object Model design pattern with Pytest, and integrated with GitHub Actions for continuous integration.

## Tech Stack
- Python
- Selenium WebDriver
- Pytest
- GitHub Actions (CI/CD)

## Features
- Page Object Model architecture
- Automated test execution on every push via GitHub Actions
- HTML test reports generated automatically
